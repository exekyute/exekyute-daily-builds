# Crossword Generator

A crossword puzzle generator that runs in the browser. It is a single HTML file with no dependencies, no build step, and no network access.

Play it here: https://exekyute.github.io/exekyute-daily-builds/miscellaneous-projects/crossword-generator/

## Features

- Three difficulty levels: Kids, Teens, and Adults
- Several topics per level
  - Kids: Animals, Colors, Food, School
  - Teens: Sports, Music, Science, Technology
  - Adults: Geography, History, Cinema, Cooking, Medical
- Each puzzle is generated by placing words on a grid so they cross at shared letters
- A Regenerate button builds a new puzzle from the same topic
- Check marks incorrect cells, Reveal fills in the answers, and Clear empties the grid
- Clue list split into Across and Down, numbered to match the grid
- Click a cell or a clue to highlight a word, type to fill it, and use arrow keys and backspace to move around
- Grid size adapts to the screen so it fits without horizontal scrolling, including on phones

## Usage

Open `index.html` in any modern web browser. Pick a difficulty and a topic, then solve.
